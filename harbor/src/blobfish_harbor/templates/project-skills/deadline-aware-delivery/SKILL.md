# Deadline-Aware Delivery

Use this workflow when remaining time is low or when progress exists but the task is still incomplete.

1. State the current best artifact, candidate, or measured result.
2. Write the best current valid output artifact now, even if you may improve it later.
3. Run the closest authoritative verification available.
4. If verification fails, make the smallest change that addresses the failure and verify again.
5. Do not open a new broad investigation branch unless the current path is clearly blocked.

Use this skill to turn partial progress into a delivered result before timeout.
